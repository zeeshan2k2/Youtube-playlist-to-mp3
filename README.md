# Youtube-playlist-to-mp3.
this program will convert youtube playlist to mp3 and store it in a file. For now this code only works on mac os I'm working on a code that'll work on windows


1. download adguard.crx the file is given in the repository

2. download chrome driver (important : note that the chrome driver version you download matches the version of the chrome you've installed on your mac) just google chrome driver and you'll be able to download it.

3. copy main.py code and make sure the packages are installed.

4. change the path in line 82 -- (os.chdir("paste your downloads folder path here") 


save all the files mentioned above in a folder.
